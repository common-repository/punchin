Easily Manage Your Employee Time Cards

== Description ==

Easily Manage Your Employee Time Cards using the punchIN plugin. punchIN allows your employees to log their hours from the wordpress dashboard. Just create a wp login for the employee, create a timecard, and start tracking their time. 

Track Employee Hours Whether You Have 1 or 1,000,001 Employees
Easy One Click Punch In / Punch Out & Publish

== Installation ==

1. Upload punchIN to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
